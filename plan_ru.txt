Задание для себя:

1. Создать новый пустой открытый репозиторий на гитхабе.
2. Создать локальный репозиторий на моем компе.
3. В локальный репозиторий положить данный файл. Добавить в отслеживание (заэдить), закоммитить. 
4. Создать файл object1.txt в качестве первого объекта тренировки, наполнить его любым текстом. Заэддить, закоммитить.
5. Создать тестовую директорию object_dir, в ней создать файл object2.txt с любым текстом. Заэдить, закоммитить.
6. Запушить репозиторий на гитхаб без использования SSH-ключа (и в дальнейшем до особого упоминания - без ключа).
7. Создать .гитигнор, создать файл object_ig.txt, директорию object_dir_ig и сразу добавить их в гитигнор. Изменить текст в object1.txt. Заэддить, закоммитить.
8. Создать ключ SSH, добавить его на гитхаб, запушить пару раз изменения (текст файлов object1.txt и object2.txt) с использованием ключа - без ввода пароля.
9. Удалить ключ с гитхаба, убедиться, что теперь снова нужен пароль, далее не использовать ключ без необходимости.
10. Создать файл object_post_ig.txt с произвольным текстом, заэдить, закоммитить, запушить.
11. Добавить файл object_post_ig.txt в .гитигнор, эддить-коммитить-пушить.
12. Убедиться, что в удаленном репозитории он не исчез.
13. Постараться, чтобы он таки исчез в удалённом репозитории. Если не получится сразу - ударить его каким-нибудь костылем :)
14. Описать решение проблемы в solution_of_problem_post_ignore.txt
15. Создать файл to_delete.txt с произвольным текстом. Заэдидить, закоммитить, запушить.
16. Удалить его мягко.
17. Поменять текст в файле to_delete.txt.
18. Создать файл partner_to_delete.txt. Заэдить его с использованием стандартного ленивого "add --all". Закоммитить. Запушить.
19. Выпулить проект и убедиться, что файл to_delete.txt таки обновлен. Не каждый сможет создать себе проблему и обрадоваться этому :). 
20. Жостка удалить to_delete.txt (в качестве костыля - вручную скопировать его во внешнюю папку, жОстка па-беларуску удалить его из проекта, скопировать его в обратный зад и добавить его в гитигнор.). Заэдидь, закоммитить, запушить.
21. Внутренне собраться, чтобы приготовиться создать себе долбанную проблему с "детатчед хед". Собрался? Вперёд!
22. Создать файл gemor1.txt с произвольным текстом. Эдить - коммитить - пушить.
23. Создать для надежности еще файл gemor2_to_delete.txt с произвольным текстом. Эдить - коммитить - пушить.
24. Чекаутить на локале предпоследний коммит.
25. Добавить файл gemor3.txt. Попробовать заэдидь-закоммитить-запушить. Словил ошибку? Маладэц! :)
26. Решить проблему. В результате должно оказаться и на локале и на удаленке gemor1.txt и gemor3.txt. Гемора 2 быть не должно. 
27. Если решил проблему, то описать решение в detatched_head_solution1.txt и дзве бульбы гэтаму хлопцу!
28. Опять внутренне собраться бороться с детатчед хед и сопутствующим траблом. (Мы легких путей не ищем). 
29. Создать gemor4.txt, эдить, коммитить, пушить.
30. Ресетнуть коммит на локале.
31. Добавить gemor5.txt. Эдить-коммитить. Задача, чтобы гемор5 был, а гемора 4 не стало - и в локальной папке и на удаленке. Пушить. Получить проблему.
32. Решить проблему, описать решение в файле detatched_head_solution1.txt. Эдить, коммитить, пушить. Дзве бульбы гэтаму хлопцу! Не, ТРЫ бульбы гэтаму хлопцу. И яшчэ шкварки.
